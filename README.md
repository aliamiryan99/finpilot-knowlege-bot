# FinPilot Knowledge Bot

This repository is scaffolded for a small Retrieval-Augmented Generation app built with Next.js and TypeScript.

## Structure

- `app/` contains the App Router UI and `app/api/chat/route.ts`
- `components/` contains the placeholder chat UI
- `lib/` contains scaffold modules for prompts, retrieval, ingestion, and graph logic
- `docs/` contains the starter knowledge base files
- `scripts/` contains placeholder ingestion utilities
- `agentic-brain/` contains project planning and evaluation notes

## Run

```bash
npm run dev
```

Then open `http://localhost:3000`.

## Current status

The repository structure is in place, but OpenAI, Chroma, and LangGraph integration are still placeholders.
